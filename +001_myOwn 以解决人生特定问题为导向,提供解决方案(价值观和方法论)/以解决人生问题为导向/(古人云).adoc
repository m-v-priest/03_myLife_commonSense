== 人的内心表达

- 书出语多虚，虚中带有无。却向书前会，放却意中珠.

 书前: 指字外或言外。
 看书之人, 不可执着于书中字面意思，而应会得其“言外之意”。

- 说诗者，不以文害辞，不以辞害志。以意逆志，是为得之。

 逆：揣测。
 解说诗的人，不要拘于文字而误解词句，也不要拘于词句而误解诗人的本意。要通过自己读作品的感受去推测诗人的本意，这样才能真正读懂诗。
