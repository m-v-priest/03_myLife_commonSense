.一切问题的核心, 都可以归源于一个核心点: 如何控制人的人性和权力.
[%collapsible%open]
====
- 官员管理问题, 宦官问题, 外戚问题, 中朝外朝问题, 诸侯军阀问题, 党争问题, 人事斗争问题, 抗击与控制少数民族入侵问题. 本质都是"如何控制人"的问题

- 有贤而不知，知贤而不用，用贤而不委，委贤而不信，此四者古今之通患也。
有贤才, 却不知道他们的存在， +
知道了贤才的存在, 却不使用他们， +
即使用了, 却不委以重任， +
即使委以重任, 却不信任人家， +
这是很多人会犯的错误。
